# Trilha-HTML DIO - Módulo-2
Este é o segundo desafio da Trilha HTML da DIO, e traz como objetivo criar um site "quase" completo, com tudo o que foi abordado neste módulo. Os temas que deverão estar presentes no projeto são formulários, estruturação e formatação de texto, mídias e tabelas, além de outros recursos falados nas aulas!
